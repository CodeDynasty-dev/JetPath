Why You'll Love JetPath

Simplicity at Its Core

JetPath strips away the boilerplate. Get started in minutes with an intuitive API inspired by the best, focusing on clarity and ease of use. Perfect for beginners and experts alike. 


# Python Example
from jetpath import JetPath

app = JetPath()

@app.get("/")
def home():
    return {"message": "Hello, JetPath!"}

# Run with: python app.py
if __name__ == "__main__":
    app.run(port=8000)

// TypeScript/JavaScript Example
import { JetPath } from 'jetpath';

const app = new JetPath();

// Route with parameters
app.get('/hello/:name', (req) => {
  const { name } = req.params;
  return `Hello, ${name}!`;
});

app.listen(3000, () => {
  console.log('🚀 Server running on http://localhost:3000');
});
